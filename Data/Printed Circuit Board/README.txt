Visual Anomaly (VisA) dataset


The image set has been modified from the original to select a subset of images used in course exercises.  The full set is available using the link referenced below.


Original Source:

https://github.com/amazon-science/spot-diff/blob/main/


The data originally appeared in the following publication:

Zou, Y., Jeong, J., Pemula, L., Zhang, D., & Dabeer, O. (2022). SPot-the-Difference Self-Supervised Pre-training for Anomaly Detection and Segmentation. arXiv preprint arXiv:2207.14315.
